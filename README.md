# Cross-Chain Bridge Simulator (Web3)

This project simulates a simple cross-chain token bridge.

It helps beginners understand how assets move between two blockchains.

## What this project does

- Locks tokens on Chain A
- Records transfer request
- Releases tokens on Chain B
- Shows bridge history

## Why this project is useful

You will learn:
- How cross-chain bridges work
- How token locking and releasing is done
- How multi-chain apps are structured
- How to track bridge transfers

## Tech used

- JavaScript
- Node.js
- Bridge concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `bridge.js` → Bridge logic  
- `chainA.js` → Source chain logic  
- `chainB.js` → Target chain logic  
- `transfer.js` → Transfer records  
- `wallet.js` → Wallet validation  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `sample.env` → Config example  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use this for real token transfers.
